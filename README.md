# LianLianKan
Windows平台上的连连看小游戏

## 项目说明
使用第三方图形库EasyX实现的连连看小游戏，使用VS2010开发，实现了基本的游戏开始界面，游戏运行界面，和游戏完成界面
连连看实现了随机地图生成保证游戏有解，并实现逻辑判断两个方块是否可消除
